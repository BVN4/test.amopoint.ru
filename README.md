Использован cron (для OpenServer):
```shell
20 4 * * * %progdir%\modules\php\%phpdriver%\php-win.exe -c %progdir%\modules\php\%phpdriver%\php.ini -q -f %sitedir%\test.amopoint.ru\cron.php
```
```shell
php cron.php
```
